## 1. Design Vision & Principles

### 1.1 Design Philosophy

**"The Matrix meets Retro Gaming"** â€” Uma experiencia imersiva que combina a estetica cyberpunk do Matrix com a nostalgia do pixel art retro, transformando um pipeline tecnico de IA em uma jornada visual envolvente.

### 1.2 UX Principles

| Principle | Description | Application |
|-----------|-------------|-------------|
| **Transparencia Radical** | Usuario sempre sabe onde esta no pipeline | Progress bar, house status, Neo position |
| **Controle sem Friccao** | Intervencoes sao seamless, nao bloqueantes | Approval popup com preview, edicao inline |
| **Feedback Imediato** | Toda acao tem resposta visual em < 500ms | Glow, particles, animations, toasts |
| **Celebracao Progressiva** | Cada conquista e recompensada | Artifact collect, stage complete, victory |
| **Imersao Contextual** | Efeitos visuais reagem ao estado do pipeline | Rain intensity, house glow, Neo aura |

### 1.3 User Persona

**Nome:** Consultor de IA (Sancho)
**Contexto:** Prepara propostas comerciais para clientes corporativos
**Necessidade:** Reduzir tempo de 4-6h para 30-60min
**Frustracao:** Processo manual repetitivo, inconsistente
**Desejo:** Ferramenta profissional que tambem seja divertida de usar

### 10.2 Motion & Animation

- **"Reduzir Animacoes" toggle** in Settings
- When enabled: disables Neo trail, glitch transitions, particle effects, code rain
- Keeps: essential status changes (color), progress updates
- Respects `prefers-reduced-motion` OS setting on first load

### 10.3 Screen Reader Support

- All interactive elements have `aria-label`
- Progress bar announces: "Etapa {n} de 7, agente {name}, status {status}"
- Toast notifications use `role="alert"` and `aria-live="assertive"`
- Approval popup uses `role="dialog"` with `aria-modal="true"`
- Agent houses have `aria-label="{name}, status: {status}"`

### 10.4 Keyboard Navigation

- All buttons and interactive elements reachable via Tab
- Focus indicators: 2px green outline (visible against dark bg)
- Focus trap in modals (approval popup, editor)
- Escape always closes active overlay

## 11. Visual Effects Specification

### 11.1 Code Rain (Canvas/Phaser)

```
 Columns: 30-40 (viewport width / 50px)
 Characters: katakana + latin + digits (random)
 Speed: 50-150px/s (random per column)

 Intensity states:
   - idle: 30 columns, 0.4 opacity
   - processing: 40 columns, 0.6 opacity
   - victory: 60 columns, 0.8 opacity, gold characters mixed
 Performance: requestAnimationFrame, skip frame if < 30fps
```
